# 排版調整任務卡 — 首頁同框 + 圖鑑分頁

> 用法：貼給 Claude Code。**這次只調版面與導覽，不動任何遊戲邏輯、卡池、IndexedDB、成長系統。**

---

## 背景

目前首頁是「三張卡在上、王國在下」，王國被擠到要往下捲動才看得到。這切斷了核心體驗：按完成時應該**同畫面立刻看到世界長出來**。要把它改成一屏同框。

（測試請用瀏覽器的**手機模擬**：Chrome F12 → 裝置工具列 → 選 iPhone。桌機視窗比例會誤導。）

---

## 要改的（6 條）

1. **對調 + 同框**：首頁改成 **王國在上、三張卡在下**，同一畫面，**不需捲動**就同時看到。
2. **鎖一屏**：首頁容器高度用 `100dvh`（動態視窗高度，避開手機網址列伸縮的坑），首頁**不可整頁捲動**。
3. **王國吃彈性高度**：王國區 `flex: 1` 吃掉上半的剩餘空間；Canvas 在該區內**等比放到最大、置中**，多餘空間延續星空背景色（看起來仍是太空）。三張卡是下方那塊，高度依內容自適應。
4. **三張卡瘦身**：縮小卡片內距與「完成」鈕的高度/留白，讓三張直向堆疊後能塞進下半部。**若放不下，優先壓縮間距與鈕高，不要讓頁面捲動。**
5. **底部導覽改成兩頁**：**首頁**（王國 + 三張卡，同框）、**圖鑑**（完成紀錄）。**移除原本獨立的「王國」分頁**——王國不該是要切過去的頁，它跟卡片同在首頁。
6. **safe-area 留白**：用 `env(safe-area-inset-top/bottom)` 留白，避開瀏海與底部手勢條。

---

## 圖鑑頁（獨立一頁）

圖鑑是「完成過的事的紀錄」，做成自己的分頁：

- 全頁顯示信心圖鑑清單（反向時間，最新在上）。
- 頂部固定顯示「已記下 N 件做到的事」。
- 清單內容多時，**只在這一頁內捲動**（首頁仍不捲動）。
- 清單資料沿用 M2 已建好的 codex，不要重做資料層。

## 驗收條件

1. 開首頁，**不捲動**就同時看到「王國（上）+ 三張卡（下）」。
2. 按任一張完成：卡片在下方拇指區好按，**成長動畫在上方同畫面同時看到**（因果不斷）。
3. 手機模擬（iPhone）下，首頁不溢出、不需捲動；瀏海與底部手勢條處有留白。
4. 底部只有「首頁 / 圖鑑」兩個分頁，**沒有獨立的「王國」頁**。
5. 切到圖鑑頁：看到完成紀錄清單與頂部總數，清單可在該頁內捲動。
6. 遊戲邏輯、卡片、成長、IndexedDB 完全沒被動到，行為跟改版前一致。
